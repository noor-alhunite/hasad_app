# حصاد - Hasad App

تطبيق Flutter متكامل لربط المزارعين بالتجار والمصانع في السوق الزراعي.

## المميزات

- **دعم ثنائي اللغة**: العربية والإنجليزية مع تبديل فوري
- **3 أنواع مستخدمين**: مزارع، تاجر، مصنع
- **تصميم متطابق مع Figma** بالألوان والتخطيط

## الشاشات

### شاشات المصادقة
- **Onboarding**: شاشة الترحيب مع اختيار نوع المستخدم
- **Login**: تسجيل الدخول برقم الجوال
- **Signup**: إنشاء حساب جديد
- **Verification**: التحقق من رقم الجوال (OTP)
- **Documents Upload**: رفع المستندات
- **Under Review**: قيد المراجعة

### شاشات المزارع
- **Dashboard (المواسم)**: لوحة التحكم الرئيسية مع الإحصائيات والمواسم النشطة
- **Add Season**: إضافة موسم زراعي جديد مع تفاصيل كاملة
- **Profile (حسابي)**: الملف الشخصي مع الإعدادات وتبديل اللغة

### شاشات مشتركة
- **Marketplace (السوق)**: السوق المباشر مع البحث والفلترة
- **AI Assistant (الذكاء)**: مساعد الذكاء الاصطناعي مع التوصيات
- **Notifications (التنبيهات)**: التنبيهات والتذكيرات والعروض
